Google+ Badge / Follow Box Widget like FB Like Box

== Description ==

Google Plus Badge Follow Box like FB Like Box which displays Followers profile pics, Follow button and +1 button.

Minimum Height and Width for the widget is 150 and maximum is 800, which is increased by 50. You can just select the height and width from drop down list.

If you don't need +1 button you can easily remove it from widget by checking remove plus one button check box in widget settings.

Additionaly you can also remove border line from widget by checking remove border check box in widget settings.

== Installation ==

1. Download the zip file and upload it using upload option in your wordpress dashbord under Plugins-> Add New.

2. Activate the plugin.

3. Goto Appearence-> widget, Now just drag and drop the Google+ Badge in your sidebar.

4. Type the ID of your Google+ page and save it.

5. That's it, Hope you enjoy the Plugin.
